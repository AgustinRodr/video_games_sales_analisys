# 🎮 Análisis de Ventas de Videojuegos

## 📊 Descripción del Proyecto

Este proyecto analiza un conjunto de datos de ventas de videojuegos hasta el año 2020 con el objetivo de identificar tendencias en la industria y aplicar modelos de Machine Learning para estimar las ventas globales.

El análisis incluye limpieza de datos, análisis exploratorio (EDA), visualización de datos y la implementación de distintos modelos de regresión para evaluar el comportamiento de las ventas.

Toda la explicación detallada del análisis se encuentra dentro del notebook.

---

## 📁 Dataset

El dataset contiene información sobre ventas de videojuegos, incluyendo:

* Nombre del videojuego
* Plataforma
* Género
* Publisher (empresa desarrolladora)
* Año de lanzamiento
* Ventas por región (Norteamérica, Europa, Japón y otras regiones)
* Ventas globales

## 🤖 Modelos de Machine Learning utilizados

En este proyecto se aplicaron distintos modelos de regresión para analizar y predecir ventas globales:

* Regresión Lineal
* Ridge Regression
* Random Forest Regression

Para evaluar los modelos se utilizaron las siguientes métricas:

* MSE (Error cuadrático medio)
* RMSE (Raíz del error cuadrático medio)
* MAE (Error absoluto medio)
* R² (Coeficiente de determinación)
